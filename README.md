# Netflix Movies and TV shows SQL Data Analysis

![Neflix Logo](https://github.com/Josh246/Netflix_sql_project/blob/main/logo.png)

## Objectives

- Analyze the distribution of contect types (Movies vs TV shows).
- Identify the most common ratings for movies and TV shows.
- List and analyze content based on release years, countries and durations.
- Explore and categorize content based on specific criteria and keywords.

## Dataset

The data for this project is sourced from the Kaggle dataset: [Netflix Movies and TV shows](https://www.kaggle.com/datasets/shivamb/netflix-shows?utm_source=chatgpt.com)
